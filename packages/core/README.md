# MCP Preflight Core

`mcp-preflight-core` is the shared scanning engine behind MCP Preflight.

Most users want the CLI package instead:

```bash
npm install -g mcp-preflight
```

Source and docs:

- Website: https://mcppreflight.com
- Repository: https://github.com/TimesAndPlaces/mcp-preflight
- Support: https://mcppreflight.com/support/
