# Quickstart Workspace

This workspace is a harmless demo that contains simulated MCP misconfigurations.

It exists so you can run `npm run quickstart` and see real MCP Preflight findings immediately, without pointing the scanner at your own repo first.

Everything in this folder is fake on purpose:
- fake credentials
- fake server names
- example-only MCP settings that are there to trigger findings

Nothing here is malicious, and nothing here should be used as a real MCP setup.
