Commercial terms

What a Pro purchase currently covers.

These terms cover MCP Preflight Pro purchases and official Pro delivery.

What you buy

What you buy

  • an official signed local license.token
  • the Pro surfaces listed in the current public documentation
  • 12 months of update entitlement from the purchase date
  • license reissue support tied to the purchase record

What stays open

What stays free

The public Lite code in the repository remains available under the repository license. A Pro purchase is not a transfer of ownership of the codebase.

What Pro is not

What Pro does not include

Pro is not a hosted service, managed backend, runtime gateway, or custom support-hours product. MCP Preflight is designed to run locally.
