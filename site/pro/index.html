CLI install

Install from the source checkout

node packages/cli/dist/index.js license install --from-file /path/to/license.token
node packages/cli/dist/index.js license status

If you use the standalone CLI bundle from GitHub Releases, run the same commands against mcp-preflight.js.

Extension install

Install from the VS Code extension

  1. Open the Command Palette.
  2. Run MCP Preflight: Install Pro License.
  3. Paste the token contents.
  4. Run MCP Preflight: Show License Status to confirm the machine is unlocked.

Reissue and recovery

Keep the token. Reissue is available if you lose it.

Keep the license.token file somewhere you can find again. If you move to another machine or lose the file, contact license support with the checkout email or Stripe payment reference.
