Security reporting

Use the private path for real product vulnerabilities.

If you find a real security issue in MCP Preflight itself, please do not post the details in a public GitHub issue.

Private path

Use the GitHub Security tab and choose Report a vulnerability.

Please include

  • what you found
  • which version you tested
  • how to reproduce it
  • any proof-of-concept material that will help confirm the issue quickly

Use public repo paths for non-sensitive feedback

  • GitHub Issues for bugs
  • GitHub Discussions for questions and feature requests
