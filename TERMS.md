# Commercial terms

These terms cover MCP Preflight Pro purchases and official Pro delivery.

## What you buy

A Pro purchase currently covers:

- an official signed local `license.token`
- the Pro workflow surfaces listed in the current public documentation
- 12 months of update entitlement from the purchase date
- license reissue support tied to the purchase record

## What stays open

The public Lite code in this repository remains available under the repository license. A Pro purchase is not a transfer of ownership of the codebase.

## What Pro is not

Pro is not a hosted account, a managed scanner backend, or a promise of custom support hours. The product is designed to run locally.

## Delivery

The current delivery path is checkout plus signed token delivery to the buyer email. Keep the token file somewhere you can recover later.

## Support

Questions, reissues, and payment help are routed through [Support](SUPPORT.md).
