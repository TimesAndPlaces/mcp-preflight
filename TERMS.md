# Terms

These terms cover MCP Preflight Pro purchases and official Pro delivery.

## What a Pro purchase includes

A Pro purchase currently includes:

- an official signed local `license.token`
- the Pro surfaces listed in the current public documentation
- 12 months of update entitlement from the purchase date
- reissue help tied to the purchase record

## What stays free

The public Lite code in this repository remains available under the repository license. Buying Pro does not transfer ownership of the codebase.

## What Pro does not include

Pro does not turn MCP Preflight into:

- a hosted account product
- a managed scanner backend
- a runtime gateway
- custom support hours

## Delivery

The current delivery path is checkout plus signed token delivery to the buyer email. Keep the token file somewhere you can recover later.
