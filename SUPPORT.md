# Support

If you need help, choose the shortest path.

## Questions and feature requests

Use [GitHub Discussions](https://github.com/TimesAndPlaces/mcp-preflight/discussions) for usage questions, workflow feedback, false-positive reports, and feature requests.

## Bug reports

Use [GitHub Issues](https://github.com/TimesAndPlaces/mcp-preflight/issues) for reproducible product bugs.

## License, payment, and reissue help

Email `support@mcppreflight.com`.

If you are asking about a Pro purchase, include the checkout email or Stripe payment reference so the purchase can be matched quickly.
